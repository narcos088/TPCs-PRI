Resumo

Criação de um servidor para tratar das ligações.

Criação de um website com índice e consequentes páginas.

Para as páginas que advém do índice, procedeu-se à leitura de ficheiros JSON.

Dado a inexistência de um "index.json", para o tratamento do índice, já foi necessário a leitura de todos os ficheiros JSON relativos às páginas secundárias, de modo a criar um objecto JSON que representasse a página principal, o índice.

A criação do objeto auxiliar que representasse o índice, foi feita no servidor.

Por fim este trabalho é realizado sobre um dataset relativo a uma coletânea de músicas de uma banda.
